# 🌿 thea
 A tool that dumps:
- Phone numbers
- Emails
- Exact addresses
- Payment information
- A little more
from a discord account. Isn't that cool?  
  
# 💾 Installation
Literally just install colored (`pip install colored`) and run the script, or if you're really lazy just run the compiled version that I've so kindly put in releases.
